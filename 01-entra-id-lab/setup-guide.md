
### 1. Entra ID Free Tenant Lab

**Folder:** `01-entra-id-lab/`

**Objective:** Simulate a corporate IAM environment using a free Microsoft Entra ID (Azure AD) tenant through the [Microsoft 365 Developer Program](https://developer.microsoft.com/en-us/microsoft-365/dev-program).

**What's included:**
- Step-by-step setup guide (`setup-guide.md`)
- User creation scripts (bulk via CSV)
- Conditional Access policy templates (`conditional-access-policies/`)
- Role assignment documentation
- SOP for Entra ID group-based access provisioning

**Skills demonstrated:**
- Microsoft Entra ID user and group management
- Conditional Access configuration
- IAM technical documentation to corporate SOP standards

**Setup:**
```bash
# No local install required — use browser-based Entra ID admin portal
# Follow setup-guide.md for tenant configuration steps
```
